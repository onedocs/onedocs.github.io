# 8.2 IDEA СИГтѕЏт╗║УЂџтљѕтиЦуеІ

1.тѕЏт╗║СИђСИфуЕ║уџё Maven жА╣уЏ«№╝џ

2.жђЅСИГтйЊтЅЇтиЦуеІ№╝їтЈ│жћ«тЇЋтЄ╗№╝їNew->Module
